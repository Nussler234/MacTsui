Attacker vs Defender Virtual Lab

Description

A hands-on cybersecurity lab project demonstrating attack and defense strategies in a controlled virtual environment.

Setup

This lab uses two virtual machines:

  • Attacker VM: Kali Linux with penetration testing tools (nmap, Metasploit, etc.)
  • Defender VM: Ubuntu with security monitoring and logging enabled

Features

  • Interactive attack simulations (port scanning, service enumeration)
  • Defense mechanism implementations (firewall rules, IDS alerts)
  • Real-world scenario walkthroughs
  • Traffic capture and analysis

Scenarios Covered

  • Network reconnaissance and mapping
  • Service vulnerability detection
  • Defense evasion techniques
  • Incident detection and response

Key Learnings

Skills: Network security, penetration testing, defensive analysis, threat detection, incident response

Getting Started

Instructions for setting up and running the lab will be added with detailed VM configurations and attack scenarios.
